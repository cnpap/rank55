ChampionListItem 召唤师技能配置测试

功能说明：
  • 点击召唤师技能图标可以打开配置弹框
  • 弹框显示当前选中的技能组合
  • 可以分别为两个技能槽选择不同的召唤师技能
  • 选中的技能会有高亮边框显示
  • 点击弹框外部区域可以关闭弹框
  • 技能变更会通过 spells-change 事件通知父组件
测试步骤：
  1. 启动开发服务器：npm run dev
  2. 在项目中找到使用 ChampionListItem 组件的页面
  3. 点击召唤师技能图标测试弹框功能
  4. 尝试选择不同的召唤师技能组合
  5. 验证点击外部关闭功能
可用的召唤师技能：
  • 净化 (ID: 1)
  • 虚弱 (ID: 3)
  • 闪现 (ID: 4) - 默认
  • 幽灵疾步 (ID: 6)
  • 治疗 (ID: 7)
  • 惩戒 (ID: 11)
  • 传送 (ID: 12)
  • 清晰术 (ID: 13)
  • 点燃 (ID: 14) - 默认
  • 屏障 (ID: 21)
  • 复活 (ID: 30)
  • 挖掘 (ID: 31)
  • 标记 (ID: 32)
组件使用示例： 
<ChampionListItem
  :champion="championData"
  :spells="[4, 14]"
  @spells-change="handleSpellsChange"
/>
